# AI Agent Sprite Simulator (Backend)

This is the Python backend for the AI Agent Sprite Simulator.

## Prerequisites

- Python 3.11

## Setup and Installation

1. **Navigate to the backend directory**:
   ```bash
   cd backend
   ```

2. **Activate the virtual environment**:
   - On **Windows**:
     ```bash
     venv\Scripts\activate
     ```
   - On **macOS/Linux**:
     ```bash
     source venv/bin/activate
     ```

3. **Install dependencies** (if adding new packages):
   ```bash
   pip install -r requirements.txt
   ```

## Running the Code

To start the backend application, ensure your virtual environment is activated, then run the main entry point:

```bash
# Replace 'main.py' with your actual starting script if it's named differently
python main.py
```
